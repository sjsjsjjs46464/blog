LRTimelapse（简称 LRT）是延时摄影后期处理的神器，尤其在处理“日转夜”（大光比变化）场景时，它是目前去闪烁和平滑过渡的最佳解决方案。本文综合了 B 站播放量前三的专业教程（如大神“Links”、“詹姆斯”等流派）的核心工作流，助您快速上手。

## 核心工作流总结 (B站高分教程精选)

LRT 的核心逻辑并非“直接编辑照片”，而是**管理元数据（Metadata）**。它与 Adobe Lightroom (LR) 配合，通过修改照片的参数文件（XMP），计算出中间的过渡数值。

### 第一阶段：LRT 初始化与关键帧

1.  **导入素材**：打开 LRT，在左侧目录树找到包含 RAW 照片序列的文件夹。LRT 会自动加载并读取元数据。
2.  **设置关键帧 (Keyframes Wizard)**：
    -   点击蓝色按钮 **"Keyframes Wizard"**。
    -   根据光线变化程度调整滑块。如果光线变化剧烈（如日落），增加关键帧数量；如果光线稳定，仅需首尾各一个。
    -   点击 **"Save"** 保存元数据。

### 第二阶段：Lightroom 调色 (LR 介入)

1.  **导入 LR**：打开 Lightroom Classic，将刚才的文件夹拖入库中（确保读取的是元数据）。
2.  **筛选关键帧**：在 LR 右下角过滤器中选择 **"LRT Keyframes"**（通常显示为 4 星或其他标记），只显示刚才在 LRT 里标记的几张照片。
3.  **调色**：
    -   对第一张关键帧进行修图。
    -   对最后一张关键帧进行修图。
    -   如果中间有其他关键帧，根据当时的光线逐一调整。
4.  **同步元数据**：选中所有关键帧，**保存元数据到文件** (`Ctrl + S` 或 `Cmd + S`)。

### 第三阶段：LRT 平滑过渡与去闪

1.  **重载数据**：回到 LRT，点击 **"Reload"** 重新读取刚才在 LR 里修改过的参数。
2.  **自动过渡 (Auto Transition)**：点击 **"Auto Transition"**，LRT 会自动计算关键帧之间每一张照片的曝光、色温等参数，生成平滑的曲线。
3.  **可视化预览 (Visual Previews)**：这是最重要的一步。点击 **"Visual Previews"**，LRT 会调用 DNG Converter 生成视频预览。
4.  **去闪烁 (Visual Deflicker)**：
    -   观察粉色曲线（亮度曲线），如果有锯齿状波动，说明有闪烁。
    -   拖动 **"Smoothing"** 滑块，点击 **"Apply"**。
    -   重复此步骤直到粉色曲线变得平滑。

### 第四阶段：导出

1.  LRT 处理完毕后，无需在 LRT 内部导出视频。
2.  回到 Lightroom，读取所有照片的元数据（Read Metadata from Files）。
3.  此时你会发现所有照片都已经按照平滑的参数调整好了。
4.  使用 **LRT Export** 插件或直接导出 JPG 序列。

## 💡 个人实战 Tips (独家经验)

在使用过程中，请务必注意以下三点关键技巧，这能帮你避开 90% 的新手坑：

### 1. 必须配合 DNG 转换器工作
LRT 并不是一个完全独立的软件，它在生成“可视化预览”（Visual Previews）这一步时，**必须依赖 Adobe DNG Converter**。
-   **注意**：请务必去 Adobe 官网下载并安装最新版的 Adobe DNG Converter（免费），否则 LRT 无法生成预览，你也无法进行可视化的去闪烁操作。

### 2. 理解 LRT 的本质：平滑过渡工具
不要把 LRT 当作剪辑软件。
-   **核心原理**：LRT 相当于给照片序列打上“标记”（关键帧），通过 LR 调好这几个标记点的颜色，然后 LRT 负责计算这两个点中间成百上千张照片的数值变化。
-   它做的是**数学题**（让参数平滑变化），而不是艺术创作。

### 3. 后期合成必须结合 AE/PR
LRT 处理完导出的通常是 JPG/TIFF 序列（或者中间格式视频）。要得到最终的成片，**必须结合视频剪辑软件**。
-   **推荐流程**：LRT 处理序列 -> 导出 JPG 序列 -> 导入 Adobe After Effects (AE) 或 Premiere Pro (PR) -> 解释素材为“图像序列” -> 添加动态模糊、配乐、变速 -> 最终渲染输出。

#### ⚠️ 额外重要提醒：关于 PR 导入 RAW 序列
这是一个很多人都会犯的错误：**Premiere Pro (PR) 无法直接导入 RAW 格式的照片序列进行编辑。**
-   **AE vs PR**：AE (After Effects) 内置了 Camera Raw 插件，可以直接读取 RAW 序列；但 PR 不行。
-   **解决方案**：在使用 PR 剪辑之前，你**必须**先在 Lightroom 中将处理好的照片导出为 **JPG** 或 **TIFF** 格式的序列，然后再拖入 PR 中进行剪辑。
